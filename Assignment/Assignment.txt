1. Command to initialize git repository - git init
2. Command to clone remote repository -  git clone <repository-url>
3. Command to stage all changes - git add -A
4. Command to view state of working directory and staging area - git status
5. Command to create new branch - git branch <branch-name>
6. Command to change branch - git checkout <branch-name>
7. Command to clean staging area - git reset HEAD -- .
8. Command to get commits and changes from remote branch - git log
9. Command to get commits and changes from remote branch (It should not merge changes in current branch). - git fetch origin <branch-name>
10. Command to get commits and changes from remote branch (It should merge changes in current branch). - git pull origin <branch-name>
11. Command to cherry pick commit which SHA => 092018283103810930 - git cherry-pick 092018283103810930
12. Command to push branch - git push origin <branch-name>
13. Command to delete remote branch - git push <remote-name> --delete <branch-name>
14. Command to do iterative rebase - git rebase <branch-name> -i
